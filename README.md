# hubot-pomme

Your #1 source of pomspiration.

See [`pomme.js`](pomme.js) for all the docs.

## Installation

In your hubot project repo, run:

`npm install hubot-pomme --save`

Then add **hubot-pomme** to your `external-scripts.json`:

```json
[
  "hubot-pomme"
]
```
